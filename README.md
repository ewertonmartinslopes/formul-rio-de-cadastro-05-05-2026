# Projeto de Cadastro

## Sistema de Cadastro de Pacientes (Tkinter)
Este projeto consiste em um sistema simples de cadastro de pacientes desenvolvido no visual studio code utilizando a biblioteca gráfica Tkinter. 
A aplicação foi projetada com foco em organização, usabilidade e separação de funcionalidades através de abas.

---

## 📌 Funcionalidades

O sistema é dividido em duas abas principais:

## 🧾 Aba 1 – Cadastro do Paciente
Permite o registro de novos pacientes por meio de um formulário com os seguintes campos obrigatórios:

Nome completo

CPF

Data de nascimento

Telefone

Email

Convenio/SUS

Contato de emergencia

Os dados inseridos são validados e armazenados para posterior visualização.

---

## 📊 Aba 2 – Pacientes Cadastrados

Exibe todos os pacientes registrados em formato de tabela, utilizando o componente Treeview do Tkinter.

Nesta aba, é possível visualizar de forma organizada todas as informações cadastradas, facilitando a consulta dos dados.
